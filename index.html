The Core Rule

When two things collide, the higher one wins
  1. Academics — class + study blocks. The only goal with deadlines you can't renegotiate. Study blocks beat everything below.
  2. JCR campaign — time-sensitive while it lasts; becomes presidency duties only if you win.
  3. Productions — demanding in bursts (rehearsal/show weeks), quiet otherwise.
  4. Design gigs — gig-based, most reschedulable. You set the deadlines, so place them in calm weeks.
